# ✅ Tasker App - Full Stack Task Management

A professional, full-stack task management application built with the **MERN Stack**. Featuring a clean UI, real-time progress tracking, and secure user authentication.

## 🚀 Live Demo
**Frontend:** [tasker-final-app.vercel.app](https://tasker-final-app.vercel.app)  
**Backend:** Deployed on Render

## ✨ Features
- **User Authentication:** Secure signup/login with JWT and profile management.
- **Task CRUD:** Create, Read, Update, and Delete tasks with ease.
- **Progress Tracking:** Interactive progress bars for each task.
- **Categorization:** Organize tasks by Work, Personal, Learning, etc.
- **Responsive Design:** Fully optimized for desktop and mobile views.

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS, Vercel
- **Backend:** Node.js, Express.js, JWT
- **Database:** MongoDB Atlas
- **State Management:** React Hooks (useState, useEffect)

## 📂 Project Structure
- `/frontend`: React application (Vercel)
- `/backend`: Node.js Express API (Render)
